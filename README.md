# ☕ Responsive Coffee Website  

Một website giới thiệu quán cà phê hiện đại, được xây dựng bằng **HTML, CSS và JavaScript**.  
Dự án tập trung vào thiết kế **responsive**, đảm bảo hiển thị tốt trên mọi thiết bị: máy tính, máy tính bảng và điện thoại.  

---

## 🚀 Công nghệ sử dụng  

- **HTML5** – cấu trúc nội dung trang web  
- **CSS3** – thiết kế giao diện, hiệu ứng responsive  
- **JavaScript (ES6)** – xử lý tương tác và hiệu ứng động  
- **ScrollReveal.js** – tạo hiệu ứng xuất hiện khi cuộn trang  
- **Remix Icon (CDN)** – bộ icon hiện đại  
- **Google Fonts** – font chữ đẹp và dễ đọc

## 📱 Tính năng nổi bật  

- Thiết kế **responsive**: tự động điều chỉnh giao diện trên mọi kích thước màn hình  
- Menu điều hướng hiện đại, dễ sử dụng  
- Hiệu ứng cuộn mượt mà với **ScrollReveal**  
- Tích hợp icon và font chữ từ nguồn mở  
- Dễ dàng triển khai trên **Netlify**
